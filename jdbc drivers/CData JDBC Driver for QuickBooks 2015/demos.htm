CData JDBC Driver for QuickBooks 2015  -  Desktop Demos

NOTE: These projects are mostly simple demos, and by no means complete applications. They are intended to illustrate the usage of the CData JDBC Driver for QuickBooks in a simple, straightforward way. What we are hoping to demonstrate is how simple it is to program with our JDBC driver. If you want to know more about it, or if you have questions, please visit www.cdata.com or email to support@cdata.com.

Java GUI Demos

Data Browser Swing Demo The Data Browser demo shows the basic features of the CData JDBC Driver for QuickBooks. After establishing a connection, you can browse the tables and get data from any table. You can also issue any SQL query to the data source. Results are displayed in a data grid, which you can use to update values and save them back to the data source.

Java Console Demos

Data Browser Console Demo The Data Browser demo shows the basic features of the CData JDBC Driver for QuickBooks. After establishing a connection, you can list the tables and get data from any table. You can also issue any SQL query to the data source.
